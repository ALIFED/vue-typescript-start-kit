## Getting start

  1. Clone this repo:
  ```
  $ git clone https://github.com/jovey-zheng/vue-typescript-start-kit.git
  ```

  2. Install dependenices:
  ```
  $ npm i
  ```

  3. Start the project:
  ```
  $ npm start               // default prot is 8080
  ```

  4. Then do whatever you want : ).
